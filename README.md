# MMedias
Programa para automaticamente calcular as notas necessarias para passar de ano
## Como usar
- Baixe o arquivo .zip
- Extraia o arquivo
- Use o Python pip para instalar as dependencias
```bash
pip install -r requirements.txt
```
- Execute o arquivo main.py
```bash
python main.py
```
